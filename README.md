# New Zealand Offender Demographics Analysis

This project examines how the age distribution of recorded offenders in
New Zealand changed between 2016 and 2024, and whether these patterns differed
across reported ethnic groups.

## Research question

How has the age distribution of unique offenders in New Zealand changed over
time, and do these trends differ across ethnic groups?

## Tools and techniques

- R
- tidyverse
- ggplot2
- Data cleaning and recoding
- Grouped summaries
- Weighted mean calculations
- Data visualisation

## Key findings

- The weighted average age of recorded offenders increased between 2016 and 2024.
- The proportion of offenders under 25 declined.
- The proportion aged 40 and over increased.
- The 25–39 age group remained the largest category.
- Similar ageing patterns appeared across the main reported ethnic groups,
  although their age distributions differed.

## Data source

New Zealand Police Unique Offenders demographic records.

## Repository files

- `offender_demographics_analysis.Rmd` — R Markdown code and analysis
- `offender_demographics_report.pdf` — completed project report

## Important note

The results describe patterns within aggregated recorded-offender data.
They should not be interpreted as population-adjusted offending rates or as
evidence that ethnicity or age causes offending.
